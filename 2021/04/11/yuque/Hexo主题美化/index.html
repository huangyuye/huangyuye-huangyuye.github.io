自动生成标题

Hexo 功能增强插件 hexo-enchancer

npm install hexo-enhancer _--save_
或
yarn add hexo-enhancer

hexo-enhancer 解析文件名的正则表达式如下:
/^.?(\d{4})[-_]?(\d{2})[-_]?(\d{2}).?[-_.@_# ]*(.*)$/_
