新建表，addColumns导入其他表中的字段，可以修改，但是两张表的修改不同步(如会员卡号，订单编号，字段类型和长度一致)

复制列 -> 共用其他表中的字段，当前表引用后不允许修改，只有被引用的表才能修改，且修改同步到所有引用到的表(如会员卡号，订单编号，字段类型和长度一致)
