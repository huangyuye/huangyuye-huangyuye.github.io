核心注解

@FeignClient

标注用于声明Feign客户端可访问的Web服务。

@EnableFeignClients

标注用于修饰Spring Boot应用的入口类，以通知Spring Boot启动应用时，扫描应用中声明的Feign客户端可访问的Web服务。
